# LMS Платформа (Обучающая система)

## Описание
LMS (Learning Management System) - это локальная веб-платформа для размещения и просмотра обучающих курсов. Система поддерживает два типа пользователей: администраторов и обычных пользователей.

## Технический стек
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript
- **Backend:** PHP 8.0+
- **База данных:** MySQL/MariaDB
- **Стиль интерфейса:** Гибрид iOS и Windows 11 с эффектами glassmorphism

## Функционал

### Для всех пользователей:
- Регистрация и авторизация
- Просмотр списка доступных курсов
- Просмотр деталей курса (видео, документы для скачивания)

### Для администраторов:
- Добавление новых курсов (название, описание, видео, дополнительные файлы)
- Удаление курсов
- Управление пользователями (просмотр, удаление)

## Установка и запуск

### Требования
- Веб-сервер (Apache/Nginx)
- PHP 8.0 или выше
- MySQL/MariaDB
- Composer (опционально)

### Шаги установки

1. **Клонирование или копирование файлов:**
   ```
   # Поместите все файлы проекта в директорию вашего веб-сервера
   # Например, в /var/www/html/lms-platform/
   ```

2. **Настройка базы данных:**
   - Откройте MySQL/MariaDB клиент
   - Выполните команды из файла `database.sql`:
     ```sql
     source /path/to/database.sql;
     ```
   - Или импортируйте файл через phpMyAdmin

3. **Настройка конфигурации:**
   - Откройте файл `config/db_config.php`
   - При необходимости измените параметры подключения к базе данных:
     ```php
     define('DB_HOST', 'localhost');      // Хост базы данных
     define('DB_NAME', 'lms_platform');   // Название базы данных
     define('DB_USER', 'root');           // Пользователь базы данных
     define('DB_PASS', '');               // Пароль пользователя
     ```

4. **Настройка прав доступа:**
   - Убедитесь, что веб-сервер имеет права на запись в папку `uploads`:
     ```bash
     chmod -R 755 uploads/
     chown -R www-data:www-data uploads/  # или другая группа веб-сервера
     ```

5. **Запуск:**
   - Откройте браузер и перейдите по адресу: `http://localhost/lms-platform/`
   - Используйте следующие учетные данные для входа администратора:
     - Логин: `admin`
     - Пароль: `admin123`

## Безопасность
- Все формы защищены от SQL-инъекций с помощью подготовленных выражений
- Пароли хэшируются с помощью функции `password_hash()`
- Реализована проверка сессий для ограничения доступа к административным функциям

## Возможные проблемы и решения

### Проблемы с загрузкой файлов:
- Проверьте права доступа к папке `uploads`
- Убедитесь, что в `php.ini` установлены соответствующие значения для `upload_max_filesize` и `post_max_size`

### Ошибки подключения к базе данных:
- Проверьте настройки подключения в `config/db_config.php`
- Убедитесь, что база данных создана и доступна
